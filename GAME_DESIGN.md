## 게임 개요

**Ball Defense**는 타워 디펜스 게임으로, 플레이어는 다양한 타워를 배치하여 경로를 따라 오는 적들을 막아야 합니다.

### 기본 설정
- **시작 골드**: 150G
- **시작 생명**: 20
- **총 웨이브**: 12
- **맵**: 3개 (S자형, 분기형, 나선형)

## 적 (Enemies) 정의

### 적 타입별 특성

#### 1. GRUNT (그런트) - 기본 보병
```
체력: 70
속도: 1.2
방어력: 0
마법저항: 0
보상: 6 골드
설명: 가장 기본적인 적. 균형잡힌 스탯.
```

#### 2. RUNNER (러너) - 고속 유닛
```
체력: 45
속도: 2.0
방어력: 0
마법저항: 0
보상: 5 골드
설명: 빠르지만 약함. 화살 타워로 빠르게 처리 필요.
```

#### 3. BRUTE (브루트) - 중장갑 탱커
```
체력: 180
속도: 0.9
방어력: 20
마법저항: 0
보상: 10 골드
설명: 높은 체력과 방어력. 마법 타워로 대응 권장.
```

#### 4. WARDED (워디드) - 마법 저항 유닛
```
체력: 132
속도: 1.1
방어력: 0
마법저항: 30
보상: 9 골드
설명: 마법 공격에 강함. 물리 타워로 대응 필요.
```

#### 5. SWARM (스웜) - 분열 유닛
```
체력: 90
속도: 1.0
방어력: 0
마법저항: 0
보상: 8 골드
특수: 죽으면 MINI로 분열
설명: 죽으면 작은 유닛으로 분열. 광역 공격 타워 필요.
```

#### 6. MINI (미니 스웜) - 분열된 유닛
```
체력: 25
속도: 1.6
방어력: 0
마법저항: 0
보상: 2 골드
설명: SWARM에서 생성. 빠르고 약함.
```

#### 7. SHIELDED (실드) - 실드 보유 유닛
```
체력: 140
속도: 1.0
방어력: 15
마법저항: 15
실드: 60
보상: 11 골드
설명: 재생 가능한 실드 보유. 지속적인 공격 필요.
```

### 웨이브 스케일링
- **체력 증가**: 웨이브당 18% (공식: 1 + 0.18 × (wave - 1))
- **보상 증가**: 웨이브당 6% (공식: 1 + 0.06 × (wave - 1))

## 게임 메커니즘

### 데미지 타입
1. **물리 데미지**: 방어력(Armor)에 의해 감소
2. **마법 데미지**: 마법저항(MR)에 의해 감소

### 데미지 계산
```
실제 데미지 = 기본 데미지 × (100 / (100 + 방어력 or MR))
```

### 특수 효과
- **크리티컬**: 일정 확률로 2배 데미지
- **슬로우**: 이동 속도 감소
- **프리즈**: 완전 정지 (스턴)
- **독 (DoT)**: 시간당 지속 데미지
- **방어력 감소**: 적의 방어력 영구 감소
- **연쇄**: 여러 적에게 데미지 전파
- **스플래시**: 범위 내 모든 적에게 데미지

### 경제 시스템
- **시작 골드**: 150G
- **적 처치 보상**: 적 타입별 상이 + 웨이브 스케일링
- **웨이브 완료 보너스**: 웨이브별 50G~300G
- **타워 판매**: 총 투자 금액의 70% 환불

### 맵 특징
1. **Map 1 (S자형)**: 긴 경로, 타워 배치 공간 넓음
2. **Map 2 (분기형)**: 복잡한 경로, 전략적 배치 필요
3. **Map 3 (나선형)**: 중앙 집중형, 범위 타워 유리

### 설치 불가 구역
- 각 맵마다 **15개의 랜덤 타일**이 설치 불가 구역으로 지정
- 빨간색 X 표시로 구분
- 매 게임마다 랜덤 생성

## 전략 가이드

### 초반 (Wave 1-3)
- Arrow Tower 2-3개로 시작
- 경로 중간과 끝에 배치
- Wave 3 전에 Frost 또는 Cannon 준비

### 중반 (Wave 4-7)
- Cannon으로 Swarm 대응
- Arcane으로 Brute 처리
- Frost로 빠른 적 감속
- 타워 업그레이드 시작

### 후반 (Wave 8-12)
- 모든 타워 Tier 2-3 업그레이드
- Poison으로 방어력 감소
- Tesla로 밀집 적 처리
- 전략적 타워 판매 및 재배치

### 타워 조합 추천
1. **물리 + 마법**: Arrow + Arcane (균형)
2. **광역 + CC**: Cannon + Frost (군중 제어)
3. **DoT + 디버프**: Poison + Tesla (지속 딜)
